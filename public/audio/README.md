Put your game audio files in this folder structure:

- `public/audio/music/menu-theme.mp3`
- `public/audio/music/game-theme.mp3`
- `public/audio/sfx/ui-click.mp3`
- `public/audio/sfx/ui-hover.mp3`
- `public/audio/sfx/attack.mp3`
- `public/audio/sfx/damage.mp3`
- `public/audio/sfx/level-up.mp3`
- `public/audio/sfx/game-over.mp3`

You can also use `.ogg` for better browser compression, but keep filenames the same
or update paths in `src/game/audio.ts`.
