# Enkelt 2D-spill med Pygame

Dette prosjektet er et enkelt 2D-spill laget med Pygame. Du styrer en firkant med piltastene, og målet er å holde deg innenfor vinduet.

## Kom i gang

1. Installer avhengigheter:
   ```
pip install pygame
   ```
2. Kjør spillet:
   ```
python main.py
   ```

## Krav
- Python 3.6+
- pygame

## Spillkontroller
- Pil opp/ned/venstre/høyre: Flytt spilleren

---

Dette er et enkelt utgangspunkt for å lage egne 2D-spill med Pygame!
